# me-tada-ta
Automatically add name, author and file location to the current InDesign document as XMP Metadata upon save

## Installation
Download 'me-tada-ta.jsx' and move to the 'startup scripts' directory inside InDesign's 'Scripts' directory. Launch or relaunch InDesign for script to take affect.

## Usage
Once installed, invoking a 'save', 'save asâ€¦', or 'export' command, either from the File menu or via a key command, will write the 'short name' of the logged in user, the name of the current InDesign document, and its location to the file's XMP metadata.
